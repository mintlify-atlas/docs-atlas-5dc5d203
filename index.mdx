Agent Skills is a collection of reusable, modular skills for Claude Code and Kiro agents. Each skill is a self-contained `SKILL.md` file that teaches your agent a specialized workflow — from writing bash scripts to following Conventional Commits to managing Terraform infrastructure. Install once via symlink, and your agent automatically applies best practices whenever relevant.

## How it works

<Steps>
  <Step title="Clone the repository">
    Clone `agent-skills` once to a local directory on your machine.
  </Step>
  <Step title="Symlink the skills you want">
    Create a symlink from each skill folder into `~/.claude/skills/` (Claude Code) or `~/.kiro/skills/` (Kiro). The agent discovers skills automatically at startup.
  </Step>
  <Step title="Your agent applies skills automatically">
    When you ask your agent to write a bash script, commit changes, or manage Terraform, it consults the matching skill and applies its guidance — no extra prompting needed.
  </Step>
  <Step title="Keep skills up to date">
    Run `git pull` inside the cloned repo. All symlinked skills update automatically.
  </Step>
</Steps>

<Tip>
  You can also install skills on a per-project basis by symlinking into `.claude/skills/` or `.kiro/skills/` inside a specific project directory.
</Tip>
